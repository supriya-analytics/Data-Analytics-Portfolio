# Book Store SQL Analysis Project

## Objective
Analyze a book store database to extract business insights using SQL.

## Tables Used
- Books
- Customers
- Orders
- Order_Details

## SQL Concepts Used
- SELECT, WHERE
- JOINs
- GROUP BY, HAVING
- Subqueries
- Aggregate functions

## Business Questions Answered
- Which books generate the highest revenue?
- Who are the top customers?
- Monthly sales trend
- Most popular book categories

## Key Insights
- Fiction category has highest sales
- Repeat customers contribute major revenue
- Sales peak during festive months

- MySQL
- SQL
